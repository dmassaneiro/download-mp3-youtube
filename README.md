1. Configure o [ffmpeg](https://ffmpeg.org/) em sua maquina, verifique se est√° conseguindo utilizar o comando `ffmpeg` no terminal
2. Execute o comando para instalar as dependencias

3. Execute o arquivo main.py
````bash
py main.py
````
